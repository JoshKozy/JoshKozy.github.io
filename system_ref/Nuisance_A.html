N2 DISPLAY IS GREEN ON MFD, BUT ENGINE DOES NOT RESPOND TO THROTTLE MOVEMENT. None None In engine start sequence, engine accelerates normally to starter cut out speed and then stagnates/hangs and N2 stabilizes between 56% and 59% N2. N2 display indicates green, but engine is stuck in start mode and will not respond to throttle movement. This nuisance condition is usually caused by selecting engine pneumatic SOV and hydraulic pumps ON before engine is able to achieve idle speed. Perform engine start per TO 1C-17A-2-71JG-00-1, 71-00-01, task 01-3 ensuring pneumatic SOV, generator and hydraulic pumps are OFF until engine is at idle. When engine start is normal, return to service with no additional action. When condition repeats, perform fault isolation per fault code 72-00-BC-_ .

SWB FFS TCAS INOP ON THE WAP OR SKE FAIL ON THE MFD WITH ASSOCIATED SWB FAULT ON THE FFS PROCESSOR The fault SWB is an internal FFS processor fault that momentarily prevents interrogation of TCAS targets.
